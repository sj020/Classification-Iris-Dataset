# Iris-Data-Set
It is a multi-variate data set.The data set consists of 50 samples from each of three species of Iris (Iris setosa, Iris virginica and Iris versicolor).
## Attributes of Iris Data-Set
* Petal Length
* Petal Width
* Sepal Length
* Sepal Width
* Species
